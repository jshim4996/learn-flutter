# learn-flutter

> AI가 가르치고, 사람이 배운다 - Flutter 앱 개발 학습 실험

---

## 프로젝트 목표

**최종 목표**: Flutter 앱 개발 → 크로스플랫폼 모바일 앱 제작

### 학습 범위
1. **Dart 기초** - 변수, 타입, 함수, 컬렉션, null safety
2. **객체지향** - 클래스, 상속, 믹스인, 추상 클래스
3. **비동기 프로그래밍** - Future, async/await, Stream
4. **Flutter 기초** - 위젯, State, 레이아웃
5. **UI 컴포넌트** - 리스트, 폼, 네비게이션
6. **상태 관리** - Provider, Riverpod
7. **네트워킹** - HTTP, 로컬 저장소
8. **인증** - Firebase Auth

---

## 학습 방식

이 프로젝트는 **AI 튜터 기반 학습 실험**입니다.

- AI가 커리큘럼에 따라 단계별로 설명
- 한 번에 하나의 항목만 학습
- 체크리스트로 진행 상황 추적
- 실행 가능한 예제 코드 제공

### 사용법
```
사용자: "오늘 어디 할 차례야"
AI: PROGRESS.md를 읽고 다음 학습 항목 안내
```
